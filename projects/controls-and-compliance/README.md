# Controls and Compliance Checklist 

## One-line summary
Reviewed a company’s security controls and compliance needs, then listed the biggest improvements to reduce risk.

## Tools used
- Controls checklist
- Compliance checklist (PCI DSS, GDPR, SOC)

## What I did
1. Checked which core controls were in place (least privilege, firewall, backups, antivirus, encryption, etc.).
2. Reviewed compliance best practices (PCI DSS, GDPR, SOC).
3. Wrote clear recommendations to close the biggest gaps.

## Recommendations I made
- Add/strengthen encryption
- Use separation of duties so everyone doesn’t have access to sensitive data
- Improve password policy
- Add backups and an IDS
